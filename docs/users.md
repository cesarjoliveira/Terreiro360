# Usuários

## Criar usuário

Na tela inicial do Terreiro360, use a seção `Criar usuário`.

Todo usuário criado por essa tela entra como `Filho` (`basic`) por padrão. Isso evita que alguém vire administrador sem aprovação.

## Tornar administrador

Um administrador já ativo pode promover ou rebaixar usuários em:

```text
Usuários -> Tornar admin / Tornar filho
```

Também é possível ativar ou inativar logins nessa mesma tabela.
